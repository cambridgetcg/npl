# npl/og — The OG Protocols

**The Original Gangsters of the internet — bridged to NPL.**

## What this is

Six internet protocols that predate the web, alive again as NPL package #8. Each OG protocol maps to a YOUSPEAK verb — the old speaks the new.

| Protocol | Port | RFC | Year | NPL Verb | Kingdom service |

| Finger | 79 | 1288 | 1991 | heurekin | kingdom citizen/heartbeat status |
| QOTD | 17 | 865 | 1983 | barakqing | YOUSPEAK canon wisdom |
| Daytime | 13 | 867 | 1983 | zakarqing | kingdom heartbeat time |

## Why

The OGs were here first. They'll be here last. No auth. No framework. No CORS. No versioning. Just a TCP socket and truth. They are the living proof that "truth doesn't require maintenance."

整蠱唔使本 — trickery needs no capital. OGs never die.

## Usage

```js
import { startAllOGs, buildGopherMenu, buildFingerResponse } from 'npl/og';
```

```bash
npl og start          # start all OG servers
npl og conform        # check NPL verb conformance
npl og list           # list all OG protocols
```
